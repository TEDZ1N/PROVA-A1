# Como funciona o código.

O programa pede uma nota ao usuário.

Ele verifica:

Se a nota é inválida (menor que 0 ou maior que 10)

Se está abaixo de 5 → Reprovado

Se está entre 5 e 6.9 → Recuperação

Se está entre 7 e 10 → Aprovado

Isso é feito usando condicionais if, elif e else.
